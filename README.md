# Portfolio - Albert Mayele

Source du portfolio personnel d'Albert Mayele. Projet construit avec Vite, React, TypeScript et Tailwind CSS.

## Développement local

Installer les dépendances et lancer le serveur de développement :

```powershell
npm install
npm run dev
```

Build pour la production et aperçu :

```powershell
npm run build
npm run preview
```

## Remarques

- Le fichier `index.html` contient les meta tags essentiels et un `site.webmanifest` minimal situé dans `public/`.
- Remplacez `public/og-image.svg` par une image OpenGraph appropriée pour un meilleur rendu lors du partage sur les réseaux sociaux.

Si vous le souhaitez, je peux préparer des configurations de déploiement pour Vercel, Netlify ou GitHub Pages.
